# Telegram бот для приема платежей через Lava

## Структура проекта
- `bot.py` — основной файл бота
- `.env` — файл с настройками (токены, ключи)
- `offer.html` — Публичная оферта
- `privacy.html` — Политика конфиденциальности
- `contacts.html` — Контакты
- `refund.html` — Политика возврата

## Настройка

### 1. Установите зависимости
```bash
pip install -r requirements.txt
```

### 2. Заполните .env файл
```
BOT_TOKEN=ваш_токен_бота
LAVA_SHOP_ID=ваш_shopId_из_Lava
LAVA_SECRET_KEY=ваш_secretKey_из_Lava
DOCS_BASE_URL=https://your-domain.com
```

### 3. Разместите документы в интернете

#### Вариант A: GitHub Pages (бесплатно)
1. Создайте репозиторий на GitHub
2. Загрузите HTML-файлы в репозиторий
3. Включите GitHub Pages в настройках репозитория
4. Используйте URL вида `https://username.github.io/repo-name/`

#### Вариант B: Любой хостинг
Загрузите HTML-файлы на ваш хостинг в одну папку.

### 4. Обновите DOCS_BASE_URL
В файле `.env` укажите URL, где размещены документы:
```
DOCS_BASE_URL=https://your-domain.com
```

### 5. Запустите бота
```bash
python3 bot.py
```

## Заполните реквизиты
В файлах `offer.html`, `contacts.html`, `privacy.html`, `refund.html` замените:
- `[ВАШИ ДАННЫЕ]` — ваше ФИО или название организации
- `[ВАШ EMAIL]` — ваш email для связи
- `[ВАШ ТЕЛЕФОН]` — ваш телефон
- `[ВАШ ИНН]` — ваш ИНН
- `[ВАШ ОГРНИП]` — ваш ОГРНИП
- `[ВАШ АДРЕС]` — ваш юридический адрес
